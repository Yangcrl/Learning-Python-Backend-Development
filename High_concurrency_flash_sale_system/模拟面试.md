## 技术问题

### 1. 索引失效场景？

**答案**：

索引失效的常见场景包括：

1. **使用不等于操作符（!=、<>）**：会导致全表扫描
2. **使用函数或表达式**：如 `WHERE YEAR(create_time) > 2023`
3. **类型转换**：如字符串与数字比较
4. **模糊查询以 % 开头**：如 `WHERE name LIKE '%abc'`
5. **OR 连接的条件**：如果其中一个条件没有使用索引，可能导致全表扫描
6. **范围查询后的数据**：范围查询（如 >、<、BETWEEN）后的字段不会使用索引
7. **NOT IN 和 NOT EXISTS**：可能导致全表扫描
8. **使用 IS NULL 或 IS NOT NULL**：可能导致全表扫描
9. **复合索引未使用最左前缀**：如复合索引 (a, b, c)，只使用 b 或 c 作为条件
10. **索引列参与计算**：如 `WHERE price * 0.8 < 100`

### 2. Redis 持久化？

**答案**：

Redis 提供两种持久化方式：

1. **RDB（Redis Database）**：
   - 原理：定期将内存中的数据快照保存到磁盘
   - 优点：文件体积小，恢复速度快
   - 缺点：可能会丢失最后一次快照后的所有数据
   - 配置：通过 `save` 命令或配置文件设置

2. **AOF（Append Only File）**：
   - 原理：将所有写操作追加到日志文件中
   - 优点：数据安全性高，最多丢失1秒的数据
   - 缺点：文件体积大，恢复速度慢
   - 配置：通过 `appendonly yes` 启用

3. **混合持久化**：
   - Redis 4.0+ 支持，结合 RDB 和 AOF 的优点
   - AOF 文件头部是 RDB 格式，后续是 AOF 格式

### 3. 分布式锁实现？

**答案**：

分布式锁的常见实现方式：

1. **基于 Redis 的实现**：
   - 使用 `SET key value NX EX seconds` 命令
   - NX：只有键不存在时才设置
   - EX：设置过期时间，防止死锁
   - 使用 Lua 脚本保证释放锁的原子性

2. **基于 ZooKeeper 的实现**：
   - 利用 ZooKeeper 的临时顺序节点
   - 每个客户端创建一个临时顺序节点
   - 检查自己是否是最小的节点，若是则获取锁
   - 否则监听前一个节点的删除事件

3. **基于数据库的实现**：
   - 使用唯一索引或悲观锁
   - 但性能较差，不适合高并发场景

4. **Redisson 分布式锁**：
   - 基于 Redis 的高级分布式锁实现
   - 支持可重入锁、公平锁、联锁等
   - 内置看门狗机制，自动续期

### 4. 缓存三大问题（穿透/击穿/雪崩）及解决方案？

**答案**：

1. **缓存穿透**：
   - 问题：请求不存在的数据，导致请求直接访问数据库
   - 解决方案：
     - 使用布隆过滤器判断数据是否存在
     - 对不存在的数据也进行缓存，设置较短的过期时间
     - 接口层验证，过滤明显无效的请求

2. **缓存击穿**：
   - 问题：热点数据的缓存过期，导致大量请求同时访问数据库
   - 解决方案：
     - 热点数据永不过期
     - 使用互斥锁，只允许一个线程去数据库查询
     - 缓存预热
     - 阶梯式过期时间

3. **缓存雪崩**：
   - 问题：大量缓存同时过期，导致所有请求访问数据库
   - 解决方案：
     - 随机过期时间
     - 多级缓存
     - 缓存预热
     - 降级策略
     - 熔断机制

### 5. 如何防止超卖？

**答案**：

防止超卖的关键是保证库存扣减的原子性：

1. **数据库层面**：
   - 使用 `UPDATE ... WHERE stock > 0` 语句
   - 使用数据库事务和锁

2. **缓存层面**：
   - 使用 Redis 预减库存
   - 使用 Lua 脚本实现原子扣减
   - 使用分布式锁

3. **业务层面**：
   - 异步处理订单
   - 最终一致性方案

## 算法问题

### 1. LeetCode 55. 跳跃游戏

**问题**：给定一个非负整数数组，你最初位于数组的第一个位置。数组中的每个元素代表你在该位置可以跳跃的最大长度。判断你是否能够到达最后一个位置。

**思路**：贪心算法，维护一个最远可达位置，遍历数组，更新最远可达位置，如果最远可达位置大于等于最后一个位置，返回true。

**代码**：

```python
def canJump(nums):
    max_reach = 0
    for i in range(len(nums)):
        if i > max_reach:
            return False
        max_reach = max(max_reach, i + nums[i])
        if max_reach >= len(nums) - 1:
            return True
    return False
```

### 2. LeetCode 45. 跳跃游戏 II

**问题**：给定一个非负整数数组，你最初位于数组的第一个位置。数组中的每个元素代表你在该位置可以跳跃的最大长度。你的目标是使用最少的跳跃次数到达数组的最后一个位置。

**思路**：贪心算法，维护当前能到达的最远位置和下一步能到达的最远位置，当到达当前最远位置时，步数加1，更新当前最远位置。

**代码**：

```python
def jump(nums):
    if len(nums) <= 1:
        return 0
    steps = 0
    current_end = 0
    next_end = 0
    for i in range(len(nums) - 1):
        next_end = max(next_end, i + nums[i])
        if i == current_end:
            steps += 1
            current_end = next_end
    return steps
```

### 3. LeetCode 62. 不同路径

**问题**：一个机器人位于一个 m x n 网格的左上角，机器人每次只能向下或者向右移动一步。机器人试图达到网格的右下角，问总共有多少条不同的路径？

**思路**：动态规划，dp[i][j]表示到达(i,j)的路径数，dp[i][j] = dp[i-1][j] + dp[i][j-1]。

**代码**：

```python
def uniquePaths(m, n):
    dp = [[1] * n for _ in range(m)]
    for i in range(1, m):
        for j in range(1, n):
            dp[i][j] = dp[i-1][j] + dp[i][j-1]
    return dp[-1][-1]
```

### 4. LeetCode 42. 接雨水

**问题**：给定 n 个非负整数表示每个宽度为 1 的柱子的高度图，计算按此排列的柱子，下雨之后能接多少雨水。

**思路**：单调栈，维护一个递减栈，当遇到比栈顶高的柱子时，计算能接的雨水。

**代码**：

```python
def trap(height):
    stack = []
    res = 0
    for i in range(len(height)):
        while stack and height[i] > height[stack[-1]]:
            top = stack.pop()
            if not stack:
                break
            distance = i - stack[-1] - 1
            bounded_height = min(height[i], height[stack[-1]]) - height[top]
            res += distance * bounded_height
        stack.append(i)
    return res
```

### 5. LeetCode 84. 柱状图中最大的矩形

**问题**：给定 n 个非负整数，用来表示柱状图中各个柱子的高度。每个柱子彼此相邻，且宽度为 1。求在该柱状图中，能够勾勒出来的矩形的最大面积。

**思路**：单调栈，维护一个递增栈，当遇到比栈顶小的柱子时，计算以栈顶为高的矩形面积。

**代码**：

```python
def largestRectangleArea(heights):
    stack = []
    max_area = 0
    heights.append(0)  # 哨兵
    for i in range(len(heights)):
        while stack and heights[i] < heights[stack[-1]]:
            h = heights[stack.pop()]
            w = i if not stack else i - stack[-1] - 1
            max_area = max(max_area, h * w)
        stack.append(i)
    return max_area
```

### 6. LeetCode 72. 编辑距离

**问题**：给定两个单词 word1 和 word2，计算出将 word1 转换成 word2 所使用的最少操作数。你可以对一个单词进行如下三种操作：插入一个字符、删除一个字符、替换一个字符。

**思路**：动态规划，dp[i][j]表示将word1的前i个字符转换为word2的前j个字符所需的最少操作数。

**代码**：

```python
def minDistance(word1, word2):
    m, n = len(word1), len(word2)
    dp = [[0] * (n + 1) for _ in range(m + 1)]
    for i in range(m + 1):
        dp[i][0] = i
    for j in range(n + 1):
        dp[0][j] = j
    for i in range(1, m + 1):
        for j in range(1, n + 1):
            if word1[i-1] == word2[j-1]:
                dp[i][j] = dp[i-1][j-1]
            else:
                dp[i][j] = min(dp[i-1][j], dp[i][j-1], dp[i-1][j-1]) + 1
    return dp[-1][-1]
```
