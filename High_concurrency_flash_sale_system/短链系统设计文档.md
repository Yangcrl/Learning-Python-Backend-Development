# 短链系统设计文档

## 1. 系统概述

短链系统是一种将长URL转换为短URL的服务，主要用于：
- 缩短社交媒体分享链接
- 跟踪链接点击统计
- 美化链接外观
- 节省短信等字符受限场景的空间

### 1.1 核心功能

- URL缩短：将长URL转换为短URL
- URL重定向：将短URL重定向到原始长URL
- 点击统计：记录链接点击次数、时间、IP等信息
- 链接管理：支持创建、查询、修改、删除短链接

## 5. 关键技术实现

### 5.1 短码生成算法

采用基于62进制的哈希算法：

1. 使用`MD5`对长URL进行哈希
2. 取哈希值的前8位
3. 将其转换为62进制字符串（0-9a-zA-Z）
4. 检查冲突，若冲突则使用下8位

```python
def generate_short_code(original_url):
    # 使用MD5哈希
    hash_value = hashlib.md5(original_url.encode()).hexdigest()
    # 转换为62进制
    chars = "0123456789abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ"
    short_code = ""
    for i in range(0, 8, 2):
        n = int(hash_value[i:i+2], 16)
        short_code += chars[n % 62]
    return short_code[:6]  # 取前6位作为短码
```

## 6. API设计

### 6.1 缩短URL

- **接口**：`POST /api/shorten`
- **请求体**：
  ```json
  {
    "original_url": "https://example.com/very/long/url/that/needs/shortening",
    "expired_at": "2026-12-31T23:59:59"
  }
  ```
- **响应**：
  ```json
  {
    "short_code": "abc123",
    "short_url": "http://short.url/abc123",
    "original_url": "https://example.com/very/long/url/that/needs/shortening",
    "created_at": "2026-04-06T16:00:00",
    "expired_at": "2026-12-31T23:59:59"
  }
  ```

### 6.2 重定向

- **接口**：`GET /{short_code}`
- **响应**：302重定向到原始URL

### 6.3 获取链接信息

- **接口**：`GET /api/links/{short_code}`
- **响应**：
  ```json
  {
    "short_code": "abc123",
    "short_url": "http://short.url/abc123",
    "original_url": "https://example.com/very/long/url/that/needs/shortening",
    "created_at": "2026-04-06T16:00:00",
    "expired_at": "2026-12-31T23:59:59",
    "click_count": 100
  }
  ```

### 6.4 获取点击统计

- **接口**：`GET /api/links/{short_code}/stats`
- **响应**：
  ```json
  {
    "short_code": "abc123",
    "total_clicks": 100,
    "daily_stats": [
      {
        "date": "2026-04-06",
        "clicks": 50
      },
      {
        "date": "2026-04-05",
        "clicks": 50
      }
    ]
  }
  ```

## 7. 部署方案

### 7.1 Docker部署

```yaml
version: '3.8'

services:
  shortlink-api:
    build: .
    ports:
      - "8000:8000"
    depends_on:
      - mysql
      - redis
    environment:
      - DATABASE_URL=mysql+pymysql://root:password@mysql:3306/shortlink
      - REDIS_URL=redis://redis:6379/0

  mysql:
    image: mysql:8.0
    environment:
      - MYSQL_ROOT_PASSWORD=password
      - MYSQL_DATABASE=shortlink
    volumes:
      - mysql_data:/var/lib/mysql

  redis:
    image: redis:6.2
    volumes:
      - redis_data:/data

volumes:
  mysql_data:
  redis_data:
```

### 7.2 环境变量配置

| 环境变量 | 描述 | 默认值 |
| --- | --- | --- |
| `DATABASE_URL` | 数据库连接URL | `mysql+pymysql://root:password@localhost:3306/shortlink` |
| `REDIS_URL` | Redis连接URL | `redis://localhost:6379/0` |
| `BASE_URL` | 短链接基础URL | `http://localhost:8000` |
| `SHORT_CODE_LENGTH` | 短码长度 | `6` |
| `CACHE_EXPIRE` | 缓存过期时间（秒） | `86400` |
